Comandos de Texto

Titulos

os comando de pahro vão de: <h1> a <h9 >tendo cada um sua utilidade
como titulo principal, titulo, sub titulo... e por ai vai

Parágrafos

tamben temo o comando: <p> ,
que é usado para esvrever paragrafos como este

voce pode quebrar um paragrafo de qualquer jeito usando a tag <BR>

Simbolos

nós podemos adicionar varios simblos usando a sintaxe: &
podendo até usar emojis com o comando &#x (id do emoji)

marcações

podemos criar uma marcação usando a tag <mark>

italico

usando a tag < EM > vocé pode deixar palavras em italico

texto grande e pequeno

podemos deixar textos pequeno usando a tag <SMALL>

texto Deletado

podemos deletar o texto usando a tag <del>

Textos Subscrito

podemos inserir textos inferiores usando a tag <SUB>

texttos Sobrescrito

podemos inserir textos superiores usando a tag <SUP>

texto Inserido

a unica forma de fazer um tesxto sublinhado é usando a tag <INS>

Citações

podemos fazer citações usando a tag <Q>

Citações completas

podemos fazer citações Completas usando a tag <BLOCKQUOTE>

para ajudar tamben podemos usar dentro desse parametro o <CITE> que ajudara em seistemas de busca pelo google

Codigos-fonte

quando citamos linhas de codigos podemos usar a tag <CODE>

junto da tag CODE podemos usar a tag <PRE> que ajuda na formatação do codigos
